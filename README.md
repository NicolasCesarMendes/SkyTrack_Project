🚁 **SkyTrack Logistics — Dashboard de Drones**
Bem-vindo ao **SkyTrack Logistics**, um protótipo de sistema logístico para gestão de drones, desenvolvido para a disciplina de **Front-End**.

---

📌 **Sobre o Projeto**
O **SkyTrack Logistics** é um dashboard criado com foco em organização, visualização e monitoramento de operações com drones.
O objetivo do sistema é apresentar uma interface moderna, responsiva e funcional para acompanhamento logístico.

Este projeto foi desenvolvido como atividade prática, aplicando conceitos fundamentais de estrutura HTML, estilização com CSS e versionamento com Git e GitHub.

---

🛠 **Tecnologias Utilizadas**
- **HTML5 Semântico**
- **CSS3** (Flexbox, Grid, Variáveis)
- **Design Responsivo** (Mobile First)
- **Git e GitHub** (Versionamento)

---

🚀 **Desafios Superados**
Durante o desenvolvimento do projeto, alguns desafios foram enfrentados e resolvidos, como:

- Implementação da **"Trade do Truncamento"** para nomes de drones
- Aplicação de **formulários com validação**
- Criação de **layout adaptável** para tablets de campo e monitores ultrawide
- Ajustes de **responsividade**
- Melhor organização do código com versionamento no GitHub
